# OpenClaw Steering Pack

This folder standardizes project steering for all customer engagements and internal builds.

## Contents
- `openclaw-steering-charter.md`: decision framework and priorities.
- `customer-project-lifecycle.md`: phase-by-phase delivery model.
- `release-gates.md`: mandatory pre-prod and release checks.
- `customer-master-plan-template.md`: reusable plan template for new customer projects.
- `github-steering-pack.md`: required GitHub markdown templates and usage.

## Usage
1. Start each customer with the master plan template.
2. Run delivery by lifecycle phase.
3. Enforce release gates before production pushes.
4. Keep PR and issue templates active in every repo.
